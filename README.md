# Todo-List-API

This is a challenge from [roadmap.sh](https://roadmap.sh) to create a simple Todo List API and consume it with a web app

## Features

- Login
- Password protection
- Security via JTW
- Full CRUD on tasks
- Tasks linked with it's users

## Requirements
- Node.js 14.x>
- Docker
- MongoDB (Atlas is recommended)
